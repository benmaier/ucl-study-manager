# AI Tool Training

In this session you will learn how to use the AI assistant that will be available to you during the upcoming tasks.

## How to use the chatbot

Click "Open AI Assistant" to open the chat interface. You can type messages and the AI will respond. The AI can also run code, create visualisations, and analyse data files.

## What to try

- Ask the AI to explain a concept (e.g. "What is a t-test?")
- Ask it to write and run a simple Python script
- Ask it to create a chart or plot
- Try uploading a file and asking questions about it

## Tips

- Be specific in your questions — the more context you provide, the better the response
- You can ask follow-up questions to refine the AI's output
- The AI can make mistakes — always verify its output critically
- You can start new conversations for different topics using the sidebar
